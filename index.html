Welcome to docker-owncloud

This script was inspired by Json Wilder's Blog and the examples found here: https://github.com/fatk/docker-letsencrypt-nginx-proxy-companion-examples. I was already using docker, owncloud, docker-gen and nginx, but with owncloud and mysql in docker and nginx with docker-get on the host. A full stack solution, with everything in docker, was the next logical step.
